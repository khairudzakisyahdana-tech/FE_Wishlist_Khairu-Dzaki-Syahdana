## âš™ï¸ Fitur Utama (CRUD)
Aplikasi ini menerapkan operasi dasar pengelolaan data:
1.  **Create (Tambah):** Menambahkan barang baru lengkap dengan harga dan catatan toko.
2.  **Read (Lihat):** Menampilkan daftar barang yang terbagi menjadi dua kategori: "Daftar Keinginan" dan "Sudah Dibeli".
3.  **Update (Ubah Status):** Menandai barang sebagai "Sudah Dibeli" (memindahkan ke rak bawah) atau mengembalikannya ke wishlist.
4.  **Delete (Hapus):** Menghapus barang dari daftar secara permanen (dilengkapi konfirmasi keamanan).
